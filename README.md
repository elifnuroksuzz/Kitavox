# 📚 Kitavox – Kişiselleştirilmiş Sesli Kitap Asistanı 🎧


> **Kitavox**, TÜBİTAK 2209-A desteği ile geliştirilmiş, **görme engelli bireyler ve sesli kitap tutkunları** için kişiselleştirilmiş bir sesli kitap asistanıdır.
> Google Text-to-Speech (TTS), yapay zekâ teknolojileri ve modern web araçlarını kullanarak metinleri **doğal ve akıcı** bir sesle okuyabilen bu proje, erişilebilirliği artırmayı ve okuma deneyimini dönüştürmeyi hedefler. 🚀

---

## ✨ Projenin Amacı

* 👩‍🦯 **Erişilebilirlik**: Görme engelli bireylerin bilgiye ve edebiyata daha rahat ulaşmasını sağlamak.
* 🎧 **Kullanıcı Deneyimi**: Basit, şık ve kolay bir arayüzle herkesin rahatlıkla kullanabileceği bir platform.
* 🔊 **Kişiselleştirme**: Farklı ses seçenekleri, hız ayarları ve dil desteği ile kullanıcıya özel bir sesli kitap deneyimi.
* 🌍 **Toplumsal Fayda**: Teknolojiyi herkes için erişilebilir kılmak ve toplumsal katkıyı artırmak.

## ⚙️ Kurulum ve Kullanım

```bash
# 1. Depoyu klonlayın
git clone https://github.com/elifnuroksuzz/Kitavox.git
cd Kitavox

# 2. Sanal ortam oluşturun
python -m venv venv

# 3. Ortamı aktif edin
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate

# 4. Gereksinimleri yükleyin
pip install -r requirements.txt

# 5. Google API anahtarınızı tanımlayın
export GOOGLE_API_KEY="sizin_api_anahtarınız"

# 6. Uygulamayı başlatın
streamlit run streamlit_app.py
```

👉 Tarayıcıda açılan arayüz üzerinden metinlerinizi girip sesli kitap olarak dinleyebilirsiniz.
